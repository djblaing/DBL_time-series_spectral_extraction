# JWST_time_series_reduction_parallel_pipeline
This repository hosts a jupyter notebook which is the initial format of Daphne Broski-Laing's custom spectral extraction procedure. This is a time series spectral extraction pipeline that uses both Eureka! and the standard jwst data processing pipeline to produce absolute flux calibrated, time series spectra. This pipeline will be released as a user-friendly package soon, but I share this jupyter notebook in the meantime.
